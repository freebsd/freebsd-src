Kionix KXCJK-1013 Accelerometer device tree bindings

Required properties:

- compatible: Must be one of:
    "kionix,kxcjk1013"
    "kionix,kxcj91008"
    "kionix,kxtj21009"
    "kionix,kxtf9"
 - reg: i2c slave address

Optional properties:

  - mount-matrix: an optional 3x3 mounting rotation matrix

Example:

kxtf9@f {
	compatible = "kionix,kxtf9";
	reg = <0x0F>;
	mount-matrix =	"0", "1", "0",
			"1", "0", "0",
			"0", "0", "1";
};
