C6X System-on-Chip
------------------

Required properties:

- compatible: "simple-bus"
- #address-cells: must be 1
- #size-cells: must be 1
- ranges

Optional properties:

- model: specific SoC model

- nodes for IP blocks within SoC


Example:

	soc {
		compatible = "simple-bus";
		model = "tms320c6455";
		#address-cells = <1>;
		#size-cells = <1>;
		ranges;

		...
	};
